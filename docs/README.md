Download `manual.html` and double-click it to open it on your computer.

To try the interactive bilingual English/Chinese crystal interface, download or open `crystal-playground.html` in the same folder and double-click it. It runs locally without a server. Use it to generate many crystal presets, upload a CIF file with symmetry expansion into the full unit cell, change lattice and atom settings, run SVGym-style edit steps, inspect the local zoom view, relax the structure, and export the observation JSON.
